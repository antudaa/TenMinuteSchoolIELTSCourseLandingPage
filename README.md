# 🧑‍🏫 10 Minute School – IELTS Course Product Page

This project replicates the IELTS course landing page from [10 Minute School](https://10minuteschool.com/product/ielts-course/) using **Next.js**, **TypeScript**, and **Tailwind CSS**. It is built to be fully **responsive**, **server-side rendered (SSR)**, and **SEO-friendly** using the official 10MS public API.

---

## 📦 Tech Stack

| Tech              | Purpose                           |
|-------------------|------------------------------------|
| **Next.js 15**    | React framework for SSR & routing |
| **React 19**      | Component-based UI                |
| **Tailwind CSS 4**| Utility-first styling             |
| **TypeScript 5**  | Type safety                       |
| **ShadCN UI**     | Accessible UI components          |
| **Radix UI**      | Headless UI primitives            |
| **Zustand**       | Global state management           |
| **React Query**   | Server-state management (API)     |
| **Axios**         | HTTP requests                     |
| **Docker**        | Containerization (optional)       |

---

## 🎯 Features

- ✅ Dynamic course data via public API
- ✅ YouTube trailer & media carousel
- ✅ Instructor showcase
- ✅ Course sections & rich descriptions
- ✅ Checklist with icons
- ✅ Bangla/English localization support (`lang=bn|en`)
- ✅ SEO meta tag support via `X-TENMS-SOURCE-PLATFORM: web` header
- ✅ Mobile & desktop carousels with thumbnails
- ✅ Server-side rendering (SSR)
- ✅ Optional: Docker-based deployment
